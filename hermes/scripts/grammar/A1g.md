## Updated A1 Grammar

### A. Sentence Foundations

1. Basic SVO word order (default neutral order)
2. Null copula in present (“Он студент.”)
3. Есть as “there is/are” (limited, mostly existential/availability; not required everywhere)
4. Simple predicate with adjective (“Она рада.” can be deferred; A1: “Она красивая.”)
5. Basic coordination with “и/а/но”
6. Basic punctuation in simple sentences (period/question mark)

### B. Yes/No Questiosn + Wh-questions

7. Yes/no questions via intonation
8. Particle “ли” for yes/no (“Ты дома, ли…?” / more natural: “Ты дома?”; still useful to model)
9. Wh-word: кто (who) + nominative pattern
10. Wh-word: что (what) + nominative/accusative pattern
11. Wh-word: где (where – static location)
12. Wh-word: куда (where to – destination)
13. Wh-word: откуда (where from) (optional A1; common)
14. Wh-word: когда (when)
15. Wh-word: почему (why) + потому что (basic)
16. Wh-word: какой/какая/какое/какие (what kind of / which) basic agreement
17. Wh-word: сколько (how many/much) + genitive plural pattern awareness

### C. Negation

18. не + verb (simple negation)
19. не + predicate adjective/noun (“Он не студент.”)
20. нет + Genitive (“У меня нет времени.”)
21. Negative answers: нет / не(т) (short answers)
22. Basic contrast: не X, а Y
